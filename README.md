# Магазин музичних інструментів

Консольна програма для управління магазином музичних інструментів.

## Опис
Система забезпечує облік товарів, клієнтів та замовлень з файловим 
зберіганням даних.

## Функціонал
- Управління товарами (CRUD, пошук, фільтрація, сортування)
- Управління клієнтами (CRUD, пошук)
- Управління замовленнями (створення, перегляд, скасування, статистика)
- Управління категоріями товарів

## Вимоги
- .NET Runtime 6.0 або вище
- Windows 7+, Linux, macOS 10.13+

## Встановлення
1. Розархівуйте архів mmi.rar
2. Перейдіть до папки mmi
3. Запустіть Магазин музичних інструментів.exe
